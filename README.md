# Feed Protocol RNG

Feed Protocol Random Number Generator(FPRNG) is a program on Solana Blockchain to derive on-chain randomness. You interact with FPRNG (making a cpi) in your program to get randomness and use in the same transaction. FPRNG charges around 1 cent in Sol. FPRNG takes advantage of the speed and security of the Solana Blockchain to offer cheap, fast, unpredictable and fair random number.
